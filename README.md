# 04-todos

04 Project from [Node de Cero a Experto](https://www.udemy.com/course/node-de-cero-a-experto)

The objective is that we create an interactive console application, with options that can be selected with the arrow keys, numbers and with the space key when there are multiple options.

Content:
- stdin
- stdout
- cycles
- Inquirer
- JavaScript classes
- JSON files
- Strongly async and await
- transformations
